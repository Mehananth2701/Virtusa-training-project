Virtusa Training Projects

 Overview

This repository is a collection of projects developed during my training, covering different programming concepts and real-world problem scenarios.
Each project focuses on a specific technology and demonstrates problem-solving, structured design, and implementation skills.

---

 Projects Included

 FareCalc – Travel Optimizer (Python)

A fare calculation system for a ride-sharing service that determines ride cost based on distance, vehicle type, and surge pricing during peak hours.

**Key Concepts:**

* Functions
* Dictionary mapping
* Conditional logic
* Input validation

---

 FinSafe – Transaction Validator (Java)

A console-based digital wallet system that validates deposits and withdrawals while preventing overdraft using exception handling.

**Key Concepts:**

* Encapsulation
* Custom Exception
* ArrayList for transaction history
* Menu-driven application

---

 Retail Insights – Stock Health Analysis (SQL)

A database-driven project that analyzes inventory performance to identify expiring products, dead stock, and revenue contribution by category.

**Key Concepts:**

* Database design
* SQL queries (JOIN, GROUP BY, SUM)
* Data analysis

---

 Resume Analyzer & Job Matcher (Python)

A tool that compares a resume with a job description and calculates a match score based on skills and keywords.

**Key Concepts:**

* NLP basics
* File handling
* Keyword extraction

---

 Quiz System (Java)

An interactive quiz application where users can attempt multiple-choice questions and receive automatic scoring.

**Key Concepts:**

* OOP concepts
* Control statements
* User interaction

---

 Hospital Management System (SQL)

A database project that manages patient, doctor, appointment, and treatment data with analytical queries.

**Key Concepts:**

* Relational database design
* SQL joins and aggregation
* Data management

---

 Technologies Used

* Python
* Java

---

 How to Use

1. Clone or download this repository
2. Navigate to any project folder
3. Follow the instructions inside each project
4. Run the code using the appropriate tool:

   * Python → VS Code / Terminal
   * Java → Compile using `javac`
   * SQL → Run in MySQL Workbench
