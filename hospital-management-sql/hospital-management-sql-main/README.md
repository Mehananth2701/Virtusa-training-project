# Hospital Management & Patient Analytics System

## Description
This project is a MySQL-based hospital management system designed to store and analyze patient, doctor, appointment, and treatment data.

## Objectives
- Manage patient records
- Store doctor details
- Track appointments
- Maintain treatment information
- Perform hospital data analysis

## Database Tables
- Patients
- Doctors
- Appointments
- Treatments

## Key SQL Operations
- Find most consulted doctors
- Calculate monthly revenue
- Identify common diseases
- Track patient visit frequency
- Analyze doctor performance

## Tools Used
- MySQL
- MySQL Workbench
- VS Code
- GitHub

## How to Run
1. Open MySQL Workbench
2. Run the `hospital_project.sql` file
3. Execute the queries to view outputs
